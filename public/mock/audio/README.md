# Mock 音频文件

将你的 MP3 文件放到这个目录下，文件名需要匹配 Mock 数据中的路径：

## 文件命名规则

| 歌曲 | 文件名 |
|------|--------|
| Lemon - 米津玄師 | `lemon.mp3` |
| 打上花火 - DAOKO × 米津玄師 | `uchiage-hanabi.mp3` |
| アイドル - YOASOBI | `idol.mp3` |
| 紅蓮華 - LiSA | `gurenge.mp3` |
| 夜に駆ける - YOASOBI | `yoru-ni-kakeru.mp3` |
| 廻廻奇譚 - Eve | `kaikai-kitan.mp3` |
| Cry Baby - Official髭男dism | `cry-baby.mp3` |
| シンデレラ - YOASOBI | `cinderella.mp3` |

## 测试用音频

如果没有真实的歌曲文件，可以使用任意 MP3 文件重命名来测试播放功能。

## 注意事项

- 文件需要是 MP3 格式
- 建议文件大小不超过 10MB
- 这些文件仅用于本地开发测试，不要提交到版本控制
